DarkIce

1. What Is DarkIce?

DarkIce is an IceCast, IceCast2 and ShoutCast live audio streamer. It takes audio input from a sound card, encodes it into mp3 and/or Ogg Vorbis, and sends the streams to one or more IceCast, ShoutCast and/or IceCast2 servers. DarkIce uses lame as a shared object as its mp3 encoder, and the Ogg Vorbis libs as its Ogg Vorbis encoder.
DarkIce is being written by Ákos Maróy

2. Status

DarkIce is under development. While still considered beta, it has been used since fall 2000 in production 24 hours per day.
The latest version is 0.14, check out the ChangeLog.
Rafael Diniz is wrting a GTK front-end for DarkIce, called DarkSnow.

5. Download

DarkIce is © Copyright Tyrell Hungary under the GNU General Public Licence, and is hosted on SourceForge.
You can download DarkIce releases from the project's download page. I also provided lame packages, which are needed by DarkIce, but you can use your own lame libs if you like.

You can either

  • Download the source tarball darkice-0.14.tar.gz and compile yourself. You will also need lame (version 3.89 or above) and Ogg Vorbis libs. I used the following libs:
    • lame-3.95.1.tar.gz
    • libogg-1.1.tar.gz
    • libvorbis-1.0.1.tar.gz
  • Get the RPM package darkice-0.14-1.i386.rpm, install & enjoy. Also install lame (version 3.89 or above) and Ogg Vorbis packages. I used the following packages:
    • lame-3.93.1-fr2.i386.rpm
    • libogg-1.1-1.i386.rpm
    • libvorbis-1.0.1-1.i386.rpm

If you like to live on the edge, you can check the CVS repository by getting module darkice as described here.
